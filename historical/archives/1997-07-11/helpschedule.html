XCF Advanced Programming Series Schedule

The classes have limited space, so advanced registration is necessary. To enroll for a particular class, go to the registration page.

Help Debugging Programs in C & C++Tuesdays 7-9pm

Tuesdays from 7-9pm will be open debugging sessions for students seeking help. Basic debugging skills will be taught and assistance in debugging programs will be provided. Bring your bugs and learn how to find them and keep them out.

Instructors : All
Prerequisites: none

Introduction to Graphical User Interface Toolkits 9/18/96 7-9pm

This class will cover some of the fundamental concepts that are common across different GUI toolkits and demonstrate a simple program, implemented in Motif, Xaw, Gtk, & Tk. The students will then implement a small program in the toolkit of their choice, while we provide assistance as necessary.

Instructors : Peter Mattis & Tracy Scott
Prerequisites: Comfortable with C & Tcl Programming

CGI Programming9/25/96 7-9pm

Interactivity on the web cannot be accomplished without CGI, or the Common Gateway Interface. In this session we will go over a few simple methods of processing and maintaining an information database. The example we will be using is the XCF helpsession signup, as well as brief discussion of the Text Science Blackboard.

Instructors : Gene Kan
Prerequisites: Perl is advised, but not required. We won't concentrate too much on the perl, rather, we'll talk more about the methods and techniques which make CGI processing a little less painful.
 Basic introduction to Java.

Instructors : Emil Moshkovich
Prerequisites: Basic Familiarity with Java

Programming in the Unix Environment 10/09/96 7-9pm

If you are curious about the internal workings of UNIX, curious about how your favorite applications work, or want to write UNIX software, this help session is for you. We will be covering: processes (learn how to start child processes and control them), file io (file descriptors and inodes, file syscalls), how to manage memory (with advanced system calls such as mmap), and interprocess communication and networking.

Instructors : Ali Rahimi
Prerequisites: None

Network Programming 10/16/96 7-9pm

Basic network programming in the Unix environment. More in depth coverage of the networking concepts introduced in the previous class.

Instructors : Ali Rahimi
Prerequisites: None

Tcl/Tk Programming 10/30/96 7-9pm

This class will cover the basics of Tcl/Tk. A quick overview of the Tcl language will be given, followed by an example of various Tk widgets. Integrating Tcl/Tk with C will also be discussed and simple applications linking with libtcl and libtk will be shown. Finally, the topic of using "tclets" (Tcl/Tk programs embedded in web browsers) will be covered, with the demonstration of a sample tclet.

Instructors : Tracy Scott
Prerequisites: None

Java 11/06/96 7-9pm

This class will provide a basic introduction to Java. More advanced Java topics will be discussed in a later session. Basic topics will include an overview of the Java Architecture, and a review of sample Java programs. You will also get a chance to ask questions you may have about Java.

Instructors : Emil Moshkovich
Prerequisites:

Advanced Java 11/13/96 7-9pm

Advanced topics in Java. Learn the in-depth issues encountered when doing serious development in Java.

Instructors : Yaroslav Faybishenko
Prerequisites: Basic Java
