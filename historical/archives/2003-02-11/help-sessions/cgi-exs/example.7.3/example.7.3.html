Example 7.3 | Server-Side generated HTML content


This is an example of a CGI script which consults a server-side file which contains extra information, indexed by product number, which is to be returned as an HTML page. This allows data to be presented on the web without pre-creating HTML files for all the data. As well, if the file is modified frequently, re-generation of those pages would be a chore, at best. This allows current data to be generated on-the-fly, as requested.
