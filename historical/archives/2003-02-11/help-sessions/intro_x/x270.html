EECS Instructional tips

The Unix machines run by EECS Instructional run something called CDE which is based on X Windows but adds a lot onto it. The biggest question people have about it is "How do I get rid of it and run regular X Windows?" To do that you need to write a file in your home directory called .dtprofile with a line like this: 

hepburn.cs% cat .dtprofile
SESSIONTYPE=xdm
