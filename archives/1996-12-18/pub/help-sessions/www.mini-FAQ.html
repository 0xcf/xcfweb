Getting Help Session handouts via the World-Wide Web
If you're using a graphical browser under Unix, start it with the URL for the help-session handouts by typing one of the following: 
	netscape http://xcf/pub/help-sessions/handouts.html
	chimera http://xcf/pub/help-sessions/handouts.html
	Mosaic http://xcf/pub/help-sessions/handouts.html
	arena http://xcf/pub/help-sessions/handouts.html
Then click on the highlighted/underlined link for the session whose handout you're interested in, then save it as a file.

If you're using a graphical browser on a personal computer choose the 'Open URL' option under the ' File' menu. Enter the following URL: 

	http://xcf/pub/help-sessions/handouts.html
Then click on the highlighted/underlined link for the session whose handout you're interested in, then save it as a file.

If you're using a textual browser, such as lynx, load the above URL by typing: 

	lynx http://xcf/pub/help-sessions/andouts.html
Use the arrow keys to move to the link for the session whose handout you're interested in and then hit 'd' to save it as a file.
